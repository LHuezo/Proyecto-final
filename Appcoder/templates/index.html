Aprende lo basico en lectura de partituras con Basics

Hola y muchas gracias por elegir Basicss como opcion para aprenedr a leer las partituras y dar tus primeros pasos para ser un pianista. Como primer paso te proporcionamos la siguiente imagen para que identificas la notas en tu piano

Como muestra la imagen las teclas blancas indican las notas naturales, mientras que las negras indican una nota sostenida o una nota bemol

Cabe destacar que habra casos en que las notas natruales no sean escritas como Do Re Mi Fa Sol La si Do, sino que se las leera como letras comunes Desde el Do como una C hasta el Si como una B ,como se se les indicara a continuacion

  1. Do = C
  2. Re = D
  3. Mi = E
  4. Fa = F
  5. Sol = G
  6. La = A
  7. Si = B
